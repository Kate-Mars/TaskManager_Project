# To-Do List App

Консольная система управления задачами с аутентификацией пользователей, отслеживанием задач и управлением сроками.

## Возможности

> **Аутентификация пользователей**  
> Регистрация и вход с хранением учетных данных
>
> **Управление задачами**  
> Создание, просмотр, редактирование и удаление задач
>
> **Отслеживание сроков**  
> Установка дедлайнов с отображением оставшегося времени
>
> **Система приоритетов**  
> Организация задач по приоритету (шкала 1-5)
>
> **Поиск и фильтрация**  
> Поиск задач по названию, описанию или приоритету
>
> **Сортировка**  
> Сортировка задач по сроку или приоритету
>
> **Сохранение данных**  
> Автоматическое сохранение в JSON-файл

## Предварительные требования

- **Java Development Kit (JDK) 8** или выше
- **Google Gson library** (для обработки JSON)

## Установка и настройка

### Шаг 1: Установка Java

1.Скачайте и установите JDK с [Oracle](https://www.oracle.com/java/technologies/javase-downloads.html) или [OpenJDK](https://openjdk.java.net/install/)
3. Проверьте установку:
```
java -version
javac -version
```

### Шаг 2: Установите библиотеку GSON

Скачайте JSON JAR с [Maven Repository](https://mvnrepository.com/artifact/com.google.code.gson/gson) или, 

Используя Maven, добавьте в ваш pom.xml:
```
<dependency>
    <groupId>com.google.code.gson</groupId>
    <artifactId>gson</artifactId>
    <version>2.10.1</version>
</dependency>
```

### Шаг 3: Настройка проекта

1. Создайте директорию проекта:
```
mkdir TaskManager
cd TaskManager
```
2. Скачайте все Java-файлы в эту директорию:

* ```Main.java```

* ```UserManager.java```

* ```User.java```

* ```TaskManager.java```

* ```Task.java```

3. Поместите JAR-файл Gson в ту же директорию или настройте classpath.

## Компиляция и запуск

### Базовая компиляция (ручной classpath)
```
# Compile all Java files
javac -cp ".:gson-2.10.1.jar" *.java

# Run the application
java -cp ".:gson-2.10.1.jar" Main
```

### Использование IDE (рекомендуется)

*IntelliJ IDEA:* 
* Создайте новый Java-проект
* Добавьте библиотеку Gson: File → Project Structure → Libraries → + → Java
* Выберите скачанный JAR-файл Gson
* Скопируйте все Java-файлы в папку src
* Запустите Main.java
  
*Eclipse:* 
* Создайте новый Java-проект
* Правой кнопкой на проект → Build Path → Configure Build Path
* Libraries → Add External JARs → Выберите JAR Gson
* Скопируйте Java-файлы в папку src
* Запустите как Java Application

## Первый запуск

### При первом запуске приложения:
1. Вы увидите меню аутентификации
2. Выберите опцию 2 для регистрации нового пользователя
3. Введите имя пользователя и пароль
4. Система автоматически создаст файл users.json
5. Начните добавлять и управлять своими задачами!

## Руководство по использованию

### Меню аутентификации
```
=== АВТОРИЗАЦИЯ ===
1. Войти
2. Зарегистрироваться
3. Выйти из приложения
```
### Главное меню
```
=== ГЛАВНОЕ МЕНЮ ===
1. Добавить задачу
2. Показать задачи
3. Удалить задачу
4. Изменить задачу
5. Сортировать задачи
6. Поиск задач
7. Выйти из аккаунта
8. Выйти из приложения
```
### Свойства задач
* __Название:__ Название задачи (обязательно) 
* __Описание:__ Дополнительная информация о задаче 
* __Срок:__ Формат ДД.ММ.ГГГГ ЧЧ:ММ 
* __Приоритет:__ Число 1-5 (1 = низкий, 5 = высокий)

### Хранение данных
* Данные пользователей автоматически сохраняются в ```users.json```
* Расположение файла: та же директория, что и приложение
* Формат: Читаемый JSON
* Резервное копирование: Рекомендуется ручное копирование для важных данных

### Поддерживаемые форматы дат
```
dd.MM.yyyy HH:mm
dd.MM.yyyy H:mm
d.M.yyyy HH:mm
d.M.yyyy H:mm
```

## При возникновении проблем с запуском
1. Убедитесь в правильности установки Java и Gson
2. Проверьте, что все файлы находятся в правильной директории
